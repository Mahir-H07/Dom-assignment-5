2. How do you select an element from the DOM?

HTML is the markup language used to structure web content, while the DOM is an object-based representation of that HTML content in the browser. When a web page loads, the browser converts HTML into the DOM, making it accessible and modifiable using JavaScript.


3. What is event delegation in the context of the DOM, and why is it useful?
